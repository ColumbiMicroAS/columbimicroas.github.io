# Rapporter
Oppgjørsrapport, K27-meldinger, fra NAV kan her lastes opp og sendes til Visma. <br />

![Reports Image](/img/docimages/payroll_plus/k27/k27-report.jpg)

## Importere K27-melding
Klikk på **Legg til fil** og velg riktig K27-melding. <br />
Når det er gjort kan du trykke på **IMPORTER FIL** for å lese innholdet inn i Payroll+. <br />

Det vil komme en logg på hvordan det gikk med importen under tabellen. <br />
![Import Log Image](/img/docimages/payroll_plus/k27/k27-report-import-log.jpg)


## Send til Visma Payroll

![Send Transaction Image](/img/docimages/payroll_plus/k27/k27-report-send-transaction1.jpg)

For å sende transaksonen til Visma Payroll velger du en eller flere linjer og trykker på *SEND TRANSAKSJON**

![Send Transaction Image](/img/docimages/payroll_plus/k27/k27-report-send-transaction2.jpg)

- Du kan velge å sende med dato fra K27-meldingen eller velge en annen dato for til og fra-dato<br />
- Du kan også velge dato for utbetalingsperoide dato (payoutPeriodDate). Denne blir sendt tom om du ikke velger noe og Visma Payroll behandler den som vanlig.<br />




## Transaksjonshistorikk
![Send Transaction Image](/img/docimages/payroll_plus/k27/k27-report-history.jpg)
På historikken kan du se hva som er overført og status på overføringen. <br />
Du kan gå inn og se detaljer på hver enkelt transaksjon ved å trykke på ![See Transaction Image](/img/docimages/payroll_plus/k27/k27-report-view.jpg). <br />

![Send Transaction Image](/img/docimages/payroll_plus/k27/k27-report-transaction-delete.jpg)
Hvis du har sendt en transaksjon til visma som du vil slette, f.eks. p.g.a. feil dato for utbetaling, så kan du slette den ved å trykke på  **SLETT TRANSAKSJON HOS VISMA**. <br

## Behandle feilet transaksjon
Noen ganger kan transaksjonen feil. Eks ved bruk av feil lønnsart eller at Visma API er nede. <br />
Du kan resette den ved ågå inn på **HISTORIKK** og trykke på **TILBAKESTILL**. <br />
![Send Transaction Image](/img/docimages/payroll_plus/k27/k27-report-failed-transaction-reset.jpg)

Du kan også slette importlinjen ved å trykke på søppelbøtta. <br />
![Send Transaction Image](/img/docimages/payroll_plus/k27/k27-report-failed-transaction-delete.jpg)
