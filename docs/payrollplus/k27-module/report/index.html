Rapporter

Oppgjørsrapport, K27-meldinger, fra NAV kan her lastes opp og sendes til Visma.

Importere K27-melding​

Klikk på Legg til fil og velg riktig K27-melding.
Når det er gjort kan du trykke på IMPORTER FIL for å lese innholdet inn i Payroll+.

Det vil komme en logg på hvordan det gikk med importen under tabellen.

Send til Visma Payroll​

For å sende transaksonen til Visma Payroll velger du en eller flere linjer og trykker på *SEND TRANSAKSJON**

  • Du kan velge å sende med dato fra K27-meldingen eller velge en annen dato for til og fra-dato
  • Du kan også velge dato for utbetalingsperoide dato (payoutPeriodDate). Denne blir sendt tom om du ikke velger noe og Visma Payroll behandler den som vanlig.

Transaksjonshistorikk​

På historikken kan du se hva som er overført og status på overføringen.
Du kan gå inn og se detaljer på hver enkelt transaksjon ved å trykke på .

Hvis du har sendt en transaksjon til visma som du vil slette, f.eks. p.g.a. feil dato for utbetaling, så kan du slette den ved å trykke på SLETT TRANSAKSJON HOS VISMA. <br

Behandle feilet transaksjon​

Noen ganger kan transaksjonen feil. Eks ved bruk av feil lønnsart eller at Visma API er nede.
Du kan resette den ved ågå inn på HISTORIKK og trykke på TILBAKESTILL.

Du kan også slette importlinjen ved å trykke på søppelbøtta.
