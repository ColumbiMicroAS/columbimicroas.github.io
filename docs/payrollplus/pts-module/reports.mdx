# Rapporter
![Reports Image](/img/docimages/payroll_plus/mpk/reports.png)

## Opprette nytt fartøy
For å opprette et nytt fartøy klikk på **OPPRETT FARTØY**.
![Ships New_Image](/img/docimages/payroll_plus/mpk/ships-new.png)

Fyll inn feltene og klikk **OPPRETT**.

Feltene  **IMO**, **Navn** og **Kj,signal** er oibligatoriske for rapporten.

Husk å sett fartøyet til **Aktiv** for at det skal genereres en rapport for det.
